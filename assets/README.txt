# Pasta de Assets

Esta pasta contém todos os recursos visuais necessários para o site:

## Arquivos Necessários
- logo.png - Logo do Thomasz Kowalski
- favicon.ico - Ícone para a guia do navegador
- hero-bg.jpg - Imagem de fundo para a seção hero
- mission.jpg - Imagem para a seção "Sobre Nossa Missão"

Certifique-se de adicionar estes arquivos para que o site funcione corretamente.

## Recomendações
- A logo.png deve ter um fundo transparente e dimensão aproximada de 150x150px
- As imagens devem ser otimizadas para web para garantir carregamento rápido
- Prefira formatos modernos como WebP quando possível, com fallbacks para JPG/PNG
